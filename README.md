# express-graphql
node + express + graphql + vue 연습용 예제

## 작업 환경
Node: v11.2.0 (v7.6 이상 사용하면 문제없을 듯)
npm: v6.11.1

## Project setup
```
# server node_modules 설치
npm install
# client node_modules 설치
cd client
npm install
```

### Server Start
```
# bash1 - root 디렉토리
npm run start
```

### Client Start
```
# bash2 - client 디렉토리
npm run serve
```
